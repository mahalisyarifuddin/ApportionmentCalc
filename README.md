# ApportionmentCalc
Electoral math, simplified.

## Introduction
ApportionmentCalc is a single-file, browser-based calculator for comparing two popular electoral seat allocation methods: **Sainte-Laguë** and **Hare Quota**. Designed for students, researchers, election observers, and anyone curious about proportional representation, this tool visualizes how different methods can produce different outcomes from the same vote counts.

The interface supports both **English** and **Bahasa Indonesia**.

## Quick Start
1. Download `ApportionmentCalc.html`.
2. Open it in any modern browser.

# HijriTrack
Tracking Hijri simultaneity for Indonesians.

## Introduction
HijriTrack is a standalone web application designed to help Indonesians track whether the start of a Hijri month is simultaneous ("serempak") across different astronomical criteria. It specifically compares the **Neo-MABIMS** criteria and the **KHGT** (Global Hijri Calendar) criteria.

## 10,000-Year Analysis
An astronomical simulation covering **10,000 Hijri years (1 AH to 10000 AH)** reveals a simultaneity rate of **69.19%**.

## Features
- **Standalone HTML:** `HijriTrack.html` works completely offline.
- **Dynamic Tracker:** Includes data for 10 years of upcoming Hijri months.
- **Bilingual:** Full support for Indonesian and English.
