# ApportionmentCalc
Matematika pemilu, disederhanakan.

## Pendahuluan
ApportionmentCalc adalah kalkulator berbasis browser untuk membandingkan dua metode alokasi kursi pemilu yang populer: **Sainte-Laguë** dan **Kuota Hare**.

# HijriTrack
Melacak keserempakan awal bulan Hijriah untuk masyarakat Indonesia.

## Pendahuluan
HijriTrack adalah aplikasi web mandiri yang dirancang untuk melacak apakah awal bulan Hijriah jatuh pada hari yang sama ("serempak") antara kriteria **Neo-MABIMS** dan **KHGT**.

## Analisis 10.000 Tahun
Simulasi astronomis mencakup **10.000 tahun Hijriah (1 H hingga 10000 H)** menunjukkan tingkat keserempakan sebesar **69,19%**.

## Fitur
- **HTML Mandiri:** `HijriTrack.html` bekerja sepenuhnya luring.
- **Pelacak Dinamis:** Mencakup data untuk 10 tahun bulan Hijriah mendatang.
- **Dua Bahasa:** Dukungan penuh untuk Bahasa Indonesia dan Inggris.
